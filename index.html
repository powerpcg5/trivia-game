JavaScript Timers Trivia Game

Enter your player names (or click on Cancel to accept the defaults); then take turns answering the timed questions using either your mouse (or other pointing device), or using your keyboard (type A—D to select answers A.—D., respectively). You have 30 seconds to answer each question. Every correctly answered question awards one point; whoever finishes the game with the most points wins the game.

Trivia questions and answers are taken from the Open Trivia Database at https://opentdb.com/

Clicking on the Settings button allows you to set the number of questions per game, the question category, the level of difficulty, the question type (multiple-choice vs. true or false), and the timeout duration for each question. These settings take effect on the next game.

The first player’s turn for the first game is chosen at random; the first player’s turn then alternates in subsequent games. After a game is complete, click on New Game to begin a new game; or optionally click on Reset All to reset everything, including player names and player scores. Good luck! :)
